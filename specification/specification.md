Specification of "roleplay"
===========================

Pages
-----

1. Character page
2. Character list
3. Combat
4. Dice
5. Resistance


###Character Page

* Icon
* Name
* Current TP
* Armor
* Weappon Skills
* Magic Lists


###Character List

* (+) Button to add a character
* List of known characters
  * modify character (goes to changeble overview page)


###Combat

* shows armor
* shows main weappon
* can dice
* slider for offensive/defensive balance
